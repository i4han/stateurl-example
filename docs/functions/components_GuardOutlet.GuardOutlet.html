Function GuardOutlet

  • GuardOutlet(): React.ReactElement | null

  • Render active guard component as an overlay

    The GuardOutlet component provides a dedicated rendering area for guard components (modals, dialogs, overlays) that appear when navigation is blocked by route guards.

    How it works:

    1. Monitors router state: Tracks routerState.activeGuard reactively
    2. Guard blocks navigation: When a guard fails, go() sets activeGuard
    3. Renders guard component: GuardOutlet detects the change and renders the component
    4. User interaction: Guard component displays modal/overlay with actions
    5. User satisfies condition: Calls next() → guard clears → navigation proceeds
    6. User cancels: Calls cancel() → guard clears → stays on current page

    Guard Component Props (GuardContext):

    Guard components receive these props:

    • intended: The destination the user tried to navigate to
    • current: The current location
    • next(): Function to satisfy the guard and proceed to next guard/navigation
    • cancel(): Function to abort the navigation entirely

    Placement:

    GuardOutlet is automatically rendered by the Router component as a sibling to the main content. You don't need to place it manually unless building a custom router.

    Returns React.ReactElement | null

    The active guard component, or null if no guard is blocking

    Example

    // Automatic usage (Router handles this internally)
    <Router routes={routes}>
      {RenderRouteMatch(match)}
      <GuardOutlet /> // ← Automatically included
    </Router>

    Example

    // Guard component receives GuardContext
    function LoginModal({ intended, current, next, cancel }: GuardContext) {
      const handleLogin = () => {
        // ... authenticate user ...
        next() // Proceed to next guard or complete navigation
      }
      
      return (
        <div className="modal">
          <h2>Login Required</h2>
          <p>You must log in to access {intended.resource}</p>
          <button onClick={handleLogin}>Login</button>
          <button onClick={cancel}>Cancel</button>
        </div>
      )
    }
