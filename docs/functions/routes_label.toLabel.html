Function toLabel

  • toLabel<L>(labelName, ...args): string

  • Convert a route label to an absolute path (without basePattern).

    Returns a path that can be used with go() to navigate. The go() function will prepend the basePattern automatically.

    Type-safe: Provides autocomplete for label names and infers param types when RouteRegistry is extended with your routes.

    Type Parameters

    • L extends never

    Parameters

    • labelName: L

      Label name defined in route config

    • Rest...args: [params?: Record<string, string | number>]

    Returns string

    Absolute path without basePattern (e.g., /users/123)

    Example

    // Navigate using label (rename-safe)
    go(toLabel('userProfile'))

    // With params - type-safe when RouteRegistry is configured
    go(toLabel('userProfile', { userId: 123 }))  // userId autocompletes

    // In click handler
    <button onClick={() => go(toLabel('settings'))}>
      Settings
    </button>

    Example

    // Enable type-safe labels by extending RouteRegistry:
    // In your app's types.d.ts:
    declare module 'stateurl' {
        interface RouteRegistry {
            routes: typeof import('./routes').routes
        }
    }

    // Now toLabel has full autocomplete:
    toLabel('loaderUser', { userId: 1 })  // ✓ userId is number
    toLabel('loaderUser', { userId: '1' })  // ✗ Type error
