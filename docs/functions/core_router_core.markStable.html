Function markStable

  • markStable(): void

  • Mark current location as stable rollback point

    This function establishes the current location as a "safe" state that the router can revert to if an error occurs or navigation fails. It's automatically called by the Router component after each successful render.

    When it's called:

    • After successful route render: Router's useEffect hook calls this once the new component has mounted without errors
    • Minimum delay: Waits at least 400ms to allow transition UI to be visible
    • Skips error pages: Won't mark error test pages as stable (they should rollback)

    What it does:

    1. Captures current location: Creates a snapshot of the full path, base, resource, query, hash
    2. Sets lastStable: Stores this snapshot as the rollback target
    3. Clears transition state: Removes the loading bar and transition indicators
    4. Clears guard state: Ensures no guard overlays are active

    Rollback mechanism:

    If an error occurs after marking stable, the rollback() function will:

    • Revert browser URL to lastStable.full
    • Update pathname signal to lastStable.full
    • Re-render the stable page

    Returns void

    Example

    // Automatically called by Router (you don't need to call this manually)
    useEffect(() => {
      const currentPath = pathname.value
      if (!isErrorPage(currentPath)) {
        setTimeout(() => markStable(), 400) // After minimum transition duration
      }
    }, [pathname.value])
