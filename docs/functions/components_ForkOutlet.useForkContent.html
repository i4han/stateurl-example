Function useForkContent

  • useForkContent(): null | ForkContent

  • useForkContent hook - Access fork content directly

    Useful when you need more control over how fork content is rendered.

    Returns null | ForkContent

    Current fork content or null

    Example

    function CustomForkRenderer() {
        const fork = useForkContent()

        if (!fork) return <EmptyState />

        return (
            <div>
                <p>Current path: {fork.breadcrumbs.join('/')}</p>
                <p>Remaining: {fork.ahead.join('/')}</p>
                {fork.content}
            </div>
        )
    }
