  • Main Router component for StateURL.

    The Router is the root component that enables URL-based navigation. It matches the current URL against route definitions and renders the corresponding components.

      Router configuration

      • Optionalbase?: string

        Static URL prefix (e.g., 'app'). Use with feature prop for cleaner syntax.

          base="app"
          feature={{ version: ['v1', 'v2'], theme: ['light', 'dark'] }}

    const routes = [
