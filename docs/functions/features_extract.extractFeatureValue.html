Function extractFeatureValue

  • extractFeatureValue(pathname, pattern, featureName): string

  • Extract a single feature value from pathname using basePattern

    NOTE: This function is kept for backward compatibility and testing. Internal feature signals now use the shared baseMatch computation to avoid duplicate parseBasePattern() calls.

    Parameters

    • pathname: string

      Current pathname (e.g., '/app/prod/2024/sales')

    • pattern: string

      Base pattern (e.g., '/app/:mode/:cycle')

    • featureName: string

      Name of feature to extract (e.g., 'mode')

    Returns string

    Feature value or '-' if not set

    Example

    extractFeatureValue('/app/prod/2024/sales', '/app/:mode/:cycle', 'mode')
    // Returns: 'prod'

    extractFeatureValue('/app/-/2024/sales', '/app/:mode/:cycle', 'mode')
    // Returns: '-' (placeholder for unset)
